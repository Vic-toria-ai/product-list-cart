# React Product Card App

This is a React project built while learning React fundamentals.

## Features

- Reusable React components
- Props and destructuring
- Product cards
- Bootstrap styling

## Technologies Used

- React
- JavaScript
- Bootstrap
- CSS

## Installation

Clone the project:

git clone https://github.com/Vic-toria-ai/product-list-cart

Install dependencies:

npm install

Run the project:

npm start
